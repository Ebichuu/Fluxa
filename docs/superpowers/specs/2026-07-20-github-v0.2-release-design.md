# Fluxa GitHub v0.2 发布设计

日期：2026-07-20

## 目标

- 将当前 Python/Flask 单后端代码发布为 Git 标签 `v0.2`。
- 使用 GitHub Actions 构建并推送 GHCR 镜像。
- 镜像同时支持 `linux/amd64` 与 `linux/arm64`。
- 发布 `ghcr.io/ebichuu/fluxa:v0.2` 与 `latest`。

## 发布边界

- `package.json` 与锁文件使用语义版本 `0.2.0`，Git 和镜像使用用户指定的短标签 `v0.2`。
- 工作流仅在 `v*` 标签或人工触发时发布，不在普通分支提交时推送镜像。
- GHCR 登录只使用 GitHub 自动提供的 `GITHUB_TOKEN`，仓库不保存个人访问令牌。
- 构建上下文排除 `.env`、运行数据、日志、缓存和参考快照。
- Compose 直接引用固定的 `ghcr.io/ebichuu/fluxa:v0.2` 镜像。

## 验证

- 本地执行 Python 回归、前端类型检查和生产构建。
- 验证 Compose 展开结果包含 GHCR `v0.2` 镜像。
- 验证 GitHub Actions YAML、Docker 构建上下文和凭据扫描。
- 推送标签后检查 Actions 结论和 GHCR 标签；失败时不把发布视为完成。
