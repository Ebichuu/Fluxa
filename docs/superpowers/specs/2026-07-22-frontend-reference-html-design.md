# 单文件前端架构参考页设计

## 目标

生成一份可直接交给其他 AI 阅读和修改的单文件前端参考页：`frontend-reference.html`。它用于解释 Fluxa 当前前端的信息架构、页面职责、状态模型、数据边界和主题系统，同时可以在浏览器中直接展示典型界面。

该文件是架构参考和静态演示，不替代 `src/` 下的 React 正式实现，不连接真实后端，不包含账号、密码、Token、Cookie 或 `.env` 内容。

## 文件边界

- 只新增一个根目录文件：`frontend-reference.html`。
- HTML、CSS、模拟数据和交互脚本全部内联，不依赖 CDN、字体服务或构建工具。
- 保持现有正式入口、React 组件、API 和 Mineradio 影院大厅不变。
- 通过 HTML 注释提供 AI 可读的架构索引，注释不展示给普通页面用户。

## 页面结构

参考页提供以下可切换的管理模块：

| 模块 | 展示内容 | 正式源码映射 |
| --- | --- | --- |
| 首页 | 今日结论、入库、下载中、待处理、需要处理、正常保护 | `Overview.tsx`、`homeSummary.ts` |
| 发现 | 榜单、媒体搜索、详情和添加追更意图 | `DiscoverPage.tsx`、`api.ts` |
| 追更 | Fluxa/Torra 对账、履约、健康三维状态和分页列表 | `DiscoverPage.tsx`、`subscriptions.ts` |
| 任务中心 | Torra → qB → 115 → Symedia → Emby 阶段链和异常解释 | `TasksCenter.tsx`、`taskChain.ts` |
| 控制室 | 服务连接、能力开关、同步和诊断 | `ControlRoom.tsx` |
| 设置 | Fluxa 账号、通知、规则偏好和界面主题 | `SettingsPage.tsx`、`RuntimeSettingsPanel.tsx` |
| 日历 | 播出日历和追更状态 | `CalendarPage.tsx` |
| 种子库 | RSS 来源、采集状态、种子搜索和匹配结果 | `RssSeedLibraryPage.tsx` |
| 影视大厅 | 独立模式入口和 Mineradio 隔离说明 | `MediaHall.tsx`、`MineradioEmbed.tsx` |

页面默认展示首页结论，并可通过顶部导航切换其他模块。影视大厅只显示隔离入口和视觉边界说明，不复制或改写 Mineradio 的实际播放 UI。

## 架构说明块

HTML 顶部保留 `ARCHITECTURE_FOR_AI` 注释，固定包含：

1. 路由到页面组件的映射。
2. 页面职责和明确不负责的内容。
3. API 分组、读取/写入边界和错误状态。
4. Fluxa/Torra 订阅对账的三维状态。
5. `mediaKey`、`targetKey`、`artifactKey`、`chainId` 四层任务身份。
6. 深色/浅色主题 Token 和影视大厅不参与主题切换的规则。
7. 参考页与正式 React 页面之间的非替代关系。

## 交互和数据

- 导航按钮切换当前模块，不改变 URL，不调用网络。
- 主题按钮在深色/浅色之间切换，并保存到 `localStorage`；无法使用存储时回退到内存状态。
- 首页、追更和任务中心使用脱敏的固定模拟数据，展示正常、等待、证据不足、正常保护和需要处理五类状态。
- “添加追更”只展示“已保存追更意图，尚未同步到 Torra”，不得写入远端。
- “重试”“暂停”等动作只显示预览说明，不执行外部动作。
- 所有按钮和状态都必须有明确的可见反馈，空列表和错误状态也要有占位内容。

## 视觉方向

- 管理页面采用当前 Fluxa 的克制深色控制台和 Apple 风格浅色磨砂玻璃双主题。
- 影视大厅保持独立、沉浸式边界，不共用管理页面的卡片布局。
- 以宽屏电脑为主要目标，使用响应式收缩保证 900px 以上仍可扫描；不为了手机堆叠所有模块。
- 使用 CSS Token 统一背景、文本、边框、状态色和阴影；浅色文字必须通过对比度检查。
- 只使用 CSS 和内联 SVG/符号图标，不加载外部资源。

## 验收标准

- 单独打开 `frontend-reference.html` 可以渲染，不需要 Vite、Flask 或网络。
- 页面只产生一个文件，不引用项目中的绝对路径和敏感配置。
- 顶部导航可以访问所有模块，主题按钮可以切换并保持状态。
- 1920×1080 下无横向溢出，顶部导航滚动时不重复或遮挡正文。
- 深浅主题下正文、辅助文字、状态标签均清晰可读。
- HTML 注释中的组件/API 映射与当前 `src/`、`docs/API_CONTRACT.md` 一致。
- 浏览器控制台无异常；不会发起真实 API、下载、订阅推送或远端删除。
