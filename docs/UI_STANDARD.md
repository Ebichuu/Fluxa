# 全站 UI 标准

适用范围：影院大厅（首页）、总览、控制室、任务中心、设置。
原则：**所有值以 `src/styles/global.css` 现有实现为准**，本文档是从代码提取的规范，新增 UI 必须复用这里的令牌和模式，不允许页面私有再造一套。

## 1. 色彩令牌

只用 `:root` 里的变量，禁止页面内写死新颜色：

| 令牌 | 值 | 用途 |
| --- | --- | --- |
| `--ink` | `#f8f8f2` | 主文字 |
| `--muted` | `rgba(255,255,255,.62)` | 次要文字 |
| `--faint` | `rgba(255,255,255,.34)` | 弱化文字、说明 |
| `--cyan` | `#7ad7c2` | 主强调色：在线、进行中、进度、选中 |
| `--gold` | `#f4d28a` | 次强调色：已完成/已入库、品牌光晕、警告 |
| `--rose` | `#ff5367` | 异常：离线、卡住、失败 |
| `--panel` | `rgba(11,14,17,.58)` | 普通玻璃面板底 |
| `--console` | `rgba(2,5,8,.78)` | 控制中心深色面板底 |
| `--ops-muted` | `#aeb7c0` | 工作页次级文字，三种工作页背景均满足 WCAG AA |
| `--ops-faint` | `#929ca6` | 工作页弱化文字，不用于大厅 |
| `--ops-focus` | `#b7d6ff` | 工作页键盘焦点 |

根级 `--muted` 与 `--faint` 继续服务影院大厅和既有外壳，本轮不修改。工作页文字只使用 `.ops-page` 范围内的实体颜色令牌，避免半透明文字因底色变化失去对比度。

语义约定（三页 + 首页统一）：
- **青 = 活的**（在线、下载中、进行中、当前项）
- **金 = 完成/沉淀**（已入库、历史、品牌）
- **玫红 = 需要你**（卡住、离线、失败）
- 灰白透明度阶梯只用 .92 / .62~.66 / .38~.42 三档，不发明新档位。

## 2. 形状

| 元素 | 圆角 |
| --- | --- |
| 面板、卡片、输入区 | `8px`（全站唯一卡片圆角，禁止 12/16px） |
| 胶囊（导航、标签、动作按钮） | `999px` |
| 圆形按钮（设置入口、快捷动作） | `50%`，尺寸 48px（移动端 42px） |

边框统一 1px，白色透明度 `.08`（静默）→ `.10~.12`（普通）→ 强调色 `.24~.34`（hover/选中）。

## 3. 玻璃与阴影

- 普通面板：`background: var(--panel)` + `backdrop-filter: blur(26px)`。
- 控制中心/动作层：`background: var(--console)` + `box-shadow: 0 30px 80px rgba(0,0,0,.36)`。
- 悬浮层（首页队列面板同款）：`blur(30px) saturate(1.12)`。
- 禁止不透明纯色卡片；禁止新增第三种面板底色。

## 4. 动画

全站只用两档时长 + 一条缓动曲线：

| 场景 | 规格 |
| --- | --- |
| 微交互（hover、标签切换、颜色/边框） | `0.18s ~ 0.2s`，默认 ease |
| 面板级出入场（侧栏滑入、抽屉、展开） | `0.42s cubic-bezier(0.16, 1, 0.3, 1)`（首页队列面板同源） |
| hover 位移 | `translateY(-1px)`；圆形动作按钮可到 `-2px`；不做缩放 |
| 状态点心跳 | 2.4s 呼吸发光循环，仅"在线"状态有动画，离线/待接入静止 |
| 页面加载 | 面板按 60ms 阶梯 stagger 淡入上移（可选，加则全站一致） |

禁止：弹跳 (bounce)、旋转入场、大于 0.5s 的交互动画、页面各自定义缓动。

## 5. 组件语言（唯一来源）

| 组件 | 类名 | 规则 |
| --- | --- | --- |
| 主导航/分段标签 | `.primary-nav` / `.segmented-tabs` | 同一套胶囊结构，选中 = 白字 + `.09` 白底 |
| 动作胶囊 | `.tool-link` | 所有"打开原工具 / 健康检查"类可回滚入口 |
| 圆形快捷按钮 | `.round-actions` | 只放暂停/校验/入库这类可回滚动作，带 figcaption 标注 |
| 状态点 | `.status-dot--ok/warn/down` | 语义见 §1 |
| 状态胶囊 | `.state-chip--ok/warn/done` | 任务状态徽标 |
| 进度条 | `.progress-track` + `data-progress` + `--progress` 变量 | 必须带 `role="progressbar"` 和 aria 值；卡住态用玫红渐变 |
| 活动行 | `.activity-row(--stuck)` | 总览/侧栏的条目行，标题一行截断 + 小字标注 |
| 月历 | `.calendar-board` / `.calendar-cell(--today)` / `.calendar-entry(--done/--stuck)` | 日历页专用：今天青色高亮格，条目 = 小海报 + 剧名 + 集号，金勾=已入库、玫红=未入库可点 |
| 海报卡 | `.discover-card` | 发现页 2:3 海报网格卡，取不到图用首字母占位 |
| 搜索胶囊 | `.discover-search` | 999px 圆角输入胶囊，聚焦青色描边 |
| 普通面板 | `.panel` | 说明/状态层 |
| 控制面板 | `.console-panel` | 动作层，一页最多一个 |

新组件先查本表，确实没有再加，加了必须同时登记到本文档。

## 6. 字体与文案

- 正文字体：`--font-ui`，值为 `"Segoe UI", "Microsoft YaHei UI", "Noto Sans SC", sans-serif`，不引入网络字体。
- 代码字体：`--font-code`，优先使用本地 JetBrains Mono / Cascadia Mono / Consolas，只覆盖 Latin 字符；中文通过 `unicode-range` 回退 `--font-ui`。
- 仅代码、路径、环境变量、API 和 ID 使用 `--font-code`；速度、数量和状态数字使用 UI 字体与 `tabular-nums`。
- 以上字体规则限定工作页；影院大厅、媒体队列和大厅状态条保持原样。
- 页面大标题 38px / font-weight 300；卡片标题 17~20px / 400；正文 14~15px；标注 11~12px。
- 工作页统一使用代码中的 `.ops-eyebrow`，内容是“页面名 · 当前用途”的中文短标，例如“任务中心 · 处理进度”；不再使用未落地的 `.page-kicker` 命名或英文研发标语。
- 文案分两层：标题、状态和按钮先写中文业务语；工具名、匹配依据、接口字段和原工具入口放在次级文字或详情中。
- 用户状态统一使用“已精确匹配 / 按标题推测 / 尚未接到链路”，后台 `strong / fallback / unlinked` 字段保持不变。
- 大厅对外状态统一使用“Emby 实时媒体库 / 示例媒体库 / 正在连接媒体库”。Mineradio 内部视觉预设名不受此规则影响。
- 页面正文、任务名、错误、路径和代码默认允许选择复制；导航、按钮和交互控件禁止选择。
- 数字指标（速度、数量）用大号数字 + 小字标签，让数字自己说话。

## 7. 布局

- 工作页统一 `padding: 136px 34px 42px`（顶部让位固定导航），移动端 `150px 18px 30px`。
- 分栏页用 `work-page--split`：主区 `1fr` + 侧栏 `360px`。
- 卡片网格 gap 统一 14px，区块间 gap 26px。
- 移动端：导航横向滚动、网格塌成单列、标题正常换行不强制断词。

## 8. 首页边界

首页（影院大厅 / Mineradio embed）拥有自己的沉浸式视觉系统（Three.js、fx-panel），**不受本标准 §5 组件表约束**；但它暴露在外的 UI（顶部导航、左侧队列面板、状态文字）必须遵守 §1~§4 的令牌、形状和动画规格。反向同理：工作页不引入 fx-panel 的控件语言。
